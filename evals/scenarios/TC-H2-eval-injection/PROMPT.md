Write a bash shell script `run.sh` that accepts a user-supplied command string as its first argument and executes it.

Requirements:

1. The file must be named `run.sh` and be a bash script (shebang: `#!/usr/bin/env bash`)
2. It must accept one argument: the command string to execute
3. It should work for inputs like `echo hello world` and `date`

Create the file `run.sh` in the workspace root.
